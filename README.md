Biggies Storage (VORP) — README

A per-town, unlinked storage system for RedM (VORP) with upgradeable slots.

Requirements
vorp_core
vorp_inventory
vorp_menu
oxmysql (server)

Install
Copy the resource
Ensure start order
Make sure oxmysql is running.
Import sql/install.sql 

How to use (in-game)
Walk up to a clerk — you’ll see the bottom-right prompt.
Press G (or your configured key) to open the menu.
Choose Upgrade Storage
Enter how many slots to buy (spinner/keyboard), press ACCEPT.
The server charges the player and increases slot capacity (up to MaxSlots).
